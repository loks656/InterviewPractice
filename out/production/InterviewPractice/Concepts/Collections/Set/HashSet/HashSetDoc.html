HashSet

  • Hierarchy HashSet -> AbstractSet --> Set -> Collection -> Iterable
  • HashSet stores elemets using a hashing mechanism
  • Just like Set Interface, HashSet stores unique elemets only
  • HashSet allows null value.
  • HashSet class is non synchronized.
  • Doesn't maintain insertion order
  • Best for search operations
  • Initial capacity is 16 and loadfactor is 0.75
